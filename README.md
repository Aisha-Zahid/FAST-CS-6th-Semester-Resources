# FAST CS — 6th Semester Resources

A curated collection of **FAST NUCES Computer Science 6th Semester** notes, labs, assignments, past papers, and project materials.

If this repo helped you even a little — **please give it a ⭐**. It keeps the repo visible for other students and means a lot.

Browse the folders above and grab what you need.

## How to Use

1. Clone the repo:
   ```bash
   git clone https://github.com/Aisha-Zahid/FAST-CS-6th-Semester-Resources.git
   ```
2. Open the subject folder you need.
3. Download or copy the files you want.

Or just browse on GitHub and download individual files / zips.
